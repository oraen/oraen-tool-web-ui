# ORAEN Tool Web UI

一个功能强大的在线工具集，提供多种实用工具的Web应用。

**在线工具**: https://tool.oraen.com/

## 项目功能

### 基础工具 (Base Utilities)
- **文本工具** - 文本处理和转换
- **数学工具** - 数学计算和转换
- **时间工具** - 时间转换和格式化
- **密码学工具** - 加密解密相关功能
- **语言工具** - 多语言支持和转换

### 文件工具 (File Utilities)
- **文件转换** - 支持多种文件格式转换
- **文件加密** - 文件加密和解密功能

### 专业工具 (Professional Tools)
- **颜色工具** - 颜色选择和格式转换
- **二维码工具** - 二维码生成和解析
- **API调试工具** - REST API测试和调试工具

### 关于
- **项目概览** - 项目介绍和开发者信息
- **其他** - 额外内容

## 项目特性

- 🎨 现代化UI设计，基于Ant Design
- ⚡ 快速响应，Vite构建优化
- 📱 响应式布局，支持多种显示模式
- 🔧 丰富的实用工具集合
- 🌙 支持深浅主题切换

## 技术栈

- **框架**: React 18.2+ with TypeScript
- **UI组件库**: Ant Design 5.9.3
- **状态管理**: Redux 4.2.1 + React-Redux 8.1.2
- **代码编辑器**: Monaco Editor
- **路由**: React Router DOM 6.16.0
- **构建工具**: Vite 3.2.7
